# Student Record Management System

## 📌 Project Overview

The Student Record Management System is a console-based application developed in **C++** using **Object-Oriented Programming (OOP)** concepts and **STL (Standard Template Library)**. It allows users to manage student records efficiently through various operations such as adding, updating, deleting, searching, sorting, filtering, and storing data.

This project was developed as part of a B.Tech CSE assignment.

---

## 🚀 Features

- Add Student Records
- Display All Students
- Search Student using Binary Search
- Update Student Details
- Delete Student Records
- Sort Students by Name
- Sort Students by Roll Number
- Sort Students by Marks
- Filter Students based on Marks
- Save Records to File
- Load Records from File

---

## 🛠 Technologies Used

- C++
- Object-Oriented Programming
- STL Vector
- STL Map
- STL Algorithms (`sort`)
- Binary Search
- File Handling (CSV)
